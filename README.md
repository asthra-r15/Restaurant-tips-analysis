# Restaurant Tips Analysis

## 📊 Project Overview
This project focuses on cleaning, analyzing, and visualizing the tips dataset.

## 🔍 Key Insights
- Saturday has highest revenue  
- Sunday has highest average tip  
- Most tips are between 2–3  

## 🛠️ Tools Used
- Python  
- Pandas  
- Seaborn
## 📈 Additional Analysis
- Created a new feature: Tip Percentage
- Found that Friday has highest average tip %
- Saturday has lowest tip %
- Visualized using bar chart
